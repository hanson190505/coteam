我们在 div 里放一个 img，发现：

在 html 和 html5 中，div 的长宽是不同的，后者的高度要超过几个像素。

比如说，下面这个是 html 的。

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Document</title>
    <style type="text/css">
      * {
        margin: 0px;
        padding: 0px;
      }
    </style>
  </head>
  <body>
    <div>
      <img src="/Users/smyhvae/Dropbox/img/20170813_1143.jpg" alt="" />
    </div>
  </body>
</html>
```
